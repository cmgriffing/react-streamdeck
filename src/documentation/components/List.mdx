# List

import { Props, Playground } from "docz";
import { SDList, SDListSelect, SDListMultiSelect } from "../../components/List";
import DemoStateContainer from "react-component-component";
import { listOptions } from '../fixtures';

## SDList (Base)

This is the base List component. Other List components wrap this one with other default config options.

### Example

<Playground>
  <div class="docz-sdpi-wrapper">
    <SDList label="List" options={listOptions} />
  </div>
</Playground>

### Props

<Props of={SDList} />

## SDListSelect

This component is a single selectable list that wraps the base component.

### Example

<Playground>
  <div class="docz-sdpi-wrapper">
    <DemoStateContainer
      initialState={{
        listState: ''
      }}
    >
      {({setState, state: { listState }}) => (
        <SDListSelect
          label="List (selectable)"
          selectedOptions={listState}
          options={listOptions}
          onChange={value => {
            console.log("I was changed.", value);
            setState({
              listState: value
            })
          }}
        />
      )}
    </DemoStateContainer>
  </div>
</Playground>

### Props

See [Base Props](#props). This wrapper passes `isSelectable: true` under the hood.
